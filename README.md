# How to setup project and run locally

### Clone the repository

`https://github.com/bitbender1010/Bridge.git`

### Install all dependencies

* Using yarn

`yarn install`

* Using npm

`npm install`

### Start watching the file and changes

Using yarn


`yarn run watch`

* Using npm


`npm run watch`


### Start project to test from the browser

* Using yarn


`yarn start`

* Using npm


`npm start`
